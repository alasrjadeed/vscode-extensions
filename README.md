# VS Code AI Services & Dev Tools Extension

A comprehensive VS Code extension providing AI-powered code analysis, linting, formatting, custom snippets, workflow automation, and testing integration.

## Features

- **AI Services**: Code analysis, bug detection, optimization suggestions
- **Code Linting**: ESLint (JS/TS), Pylint (Python)
- **Code Formatting**: Prettier, Black
- **Custom Snippets**: React, Vue, Node.js, Django, FastAPI
- **Workflow Automation**: Build pipelines, CI/CD integration
- **Testing Integration**: Jest, Pytest support
- **Problem Solving**: Error diagnosis and solutions
- **Code Upgrading**: Version migration assistance
- **Feature Templates**: Scaffolding for new features

## Quick Start

1. Install dependencies: `npm install`
2. Compile: `npm run compile`
3. Debug: Press F5 in VS Code
4. Build: `npm run build`

## Commands

| Command | Shortcut | Description |
|---------|----------|-------------|
| AI Services: Analyze Code | - | AI code analysis |
| Format: Apply Code Formatting | Ctrl+Shift+F | Format code |
| Lint: Check Code Quality | Ctrl+Shift+L | Lint code |
| Insert: Custom Code Snippet | - | Insert snippets |
| Test: Run Validation Tests | - | Run tests |
| Deploy: Trigger Build Pipeline | - | Trigger build |

## Tech Stack Support

- **JavaScript/TypeScript**: React, Vue, Node.js, Angular
- **Python**: Django, FastAPI, Flask
- **Testing**: Jest, Pytest, unittest
